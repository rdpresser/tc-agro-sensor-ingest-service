==> When running shell commands, always run from the /src folder

==> command to add a new migration
Shell: dotnet ef migrations add <migration-name> -c ApplicationDbContext -s TC.Agro.SensorIngest.Service -p TC.Agro.SensorIngest.Infrastructure
PMC: Add-Migration <migration-name> -Context ApplicationDbContext -Project TC.Agro.SensorIngest.Infrastructure -StartupProject TC.Agro.SensorIngest.Service

==> command to remove the last migration created (not in database)
Shell: dotnet ef migrations remove -c ApplicationDbContext -s TC.Agro.SensorIngest.Service -p TC.Agro.SensorIngest.Infrastructure
PMC: Remove-Migration -Context ApplicationDbContext -Project TC.Agro.SensorIngest.Infrastructure -StartupProject TC.Agro.SensorIngest.Service

==> command to update the database with all pending migrations
Shell: dotnet ef database update -c ApplicationDbContext -s TC.Agro.SensorIngest.Service -p TC.Agro.SensorIngest.Infrastructure
PMC: Update-Database -Context ApplicationDbContext -Project TC.Agro.SensorIngest.Infrastructure -StartupProject TC.Agro.SensorIngest.Service

==> command to remove all migrations from database but not the files from project
Shell: dotnet ef database update 0 -c ApplicationDbContext -s TC.Agro.SensorIngest.Service -p TC.Agro.SensorIngest.Infrastructure
PMC: Update-Database -Migration 0 -Context ApplicationDbContext -Project TC.Agro.SensorIngest.Infrastructure -StartupProject TC.Agro.SensorIngest.Service
PMC: Drop-Database -Context ApplicationDbContext -Project TC.Agro.SensorIngest.Infrastructure -StartupProject TC.Agro.SensorIngest.Service

==> command to undo to a specific migration
Shell: dotnet ef database update <migration-name> -c ApplicationDbContext -s TC.Agro.SensorIngest.Service -p TC.Agro.SensorIngest.Infrastructure
PMC: Update-Database <migration-name> -Context ApplicationDbContext -Project TC.Agro.SensorIngest.Infrastructure -StartupProject TC.Agro.SensorIngest.Service

==> command to generate the SQL Script DB for ALL database
Shell: dotnet ef migrations script -c ApplicationDbContext -s TC.Agro.SensorIngest.Service -p TC.Agro.SensorIngest.Infrastructure | out-file ./scriptFile.sql
PMC: Script-Migration <migration-name>
PMC: Script-Migration -Context ApplicationDbContext -Project TC.Agro.SensorIngest.Infrastructure -StartupProject TC.Agro.SensorIngest.Service

==>Command to revert migration with script
Script-Migration -Project TC.Agro.SensorIngest.Infrastructure -StartupProject TC.Agro.SensorIngest.Service -From <migration-name> -To 0

==> command to upgrade a specific database without having to change windows environment
PMC: $env:ASPNETCORE_ENVIRONMENT="Development"

==> Command to install EF Core Migration CLI Tool
Shell: dotnet tool install --global dotnet-ef

==> Command to update EF Core Migration CLI Tool with the latest version
Shell: dotnet tool uninstall --global dotnet-ef
Shell: dotnet tool install --global dotnet-ef

==> Command to list all dotnet Tools installed
Shell: dotnet tool list

==> Command to list all available commands for Entity Framework Core
PMC: Get-Help EntityFramework
PMC: Get-Help Add-Migration
